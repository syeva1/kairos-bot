# KAIROS Bot

Telegram бот-напоминалка с интеграцией Todoist.

## Что умеет
- Каждый час присылает невыполненные задачи из Todoist
- Кнопка ✅ — отмечает задачу выполненной
- Кнопка ➕ — создаёт новую задачу с приоритетом
- Кнопка 🔄 — обновляет список
- Команды: /tasks, /new, /start

## Переменные окружения
- `TG_BOT_TOKEN` — токен Telegram бота
- `TG_CHAT_ID` — chat_id пользователя
- `TODOIST_TOKEN` — API токен Todoist
